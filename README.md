# customer-segmentation-persona-assistant
AI-powered ecommerce customer segmentation project using K-Means clustering to analyze user behavior, create customer personas, recommend marketing actions, and provide a Gradio app for real-time segment prediction.

This project analyzes ecommerce customer behavior and groups customers into different segments using K-Means clustering. It uses customer activity data such as views, cart actions, purchases, price, brand, category, and sessions.
The project also creates customer personas and marketing recommendations for each segment. A Gradio app is included to predict customer segments based on user behavior inputs.
